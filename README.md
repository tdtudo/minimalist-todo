# ToDo_list - 用户手册

## 简介

ToDo_list 是一个极简风格的桌面待办清单应用，采用透明毛玻璃效果，始终显示在桌面上，不会干扰你的工作流程。

## 安装

### 便携版（推荐）
1. 双击 `ToDo_list 1.0.0.exe`
2. 直接运行，无需安装
3. 可以放在任何位置（桌面、U盘等）

### 安装版
1. 双击 `ToDo_list Setup 1.0.0.exe`
2. 选择安装路径
3. 点击"下一步"完成安装
4. 自动创建桌面快捷方式

## 功能说明

### 窗口控制

应用窗口左上角有三个彩色按钮：

- **红色按钮** - 关闭应用
- **黄色按钮** - 最小化窗口
- **绿色按钮** - 固定窗口到桌面顶层
  - 固定后按钮会变成紫色
  - 再次点击取消固定

### 添加任务

1. 点击右上角的 **+** 按钮
2. 在弹出的输入框中输入任务内容
3. 点击"添加"按钮或按 **Enter** 键确认
4. 点击输入框外的任意位置或再次点击 **+** 按钮关闭输入框

### 管理任务

- **完成任务** - 点击任务左侧的复选框
  - 任务会自动移到列表底部
  - 文字显示删除线和较浅的颜色
- **删除任务** - 鼠标悬停在任务上，点击右侧出现的"删除"按钮
- **查看任务** - 已完成的任务显示在列表底部

### 窗口特性

- **透明背景** - 半透明毛玻璃效果
- **不在 Alt+Tab 中显示** - 不干扰应用切换
- **不在任务栏显示** - 像桌面小部件
- **可拖动** - 按住窗口标题栏拖动位置
- **始终置顶** - 固定后显示在其他窗口之上

## 数据存储

- 所有任务数据自动保存在本地
- 数据文件位置：`%APPDATA%\my-todo-app\todos.json`
- 关闭应用后数据不会丢失

## 快捷操作

- **Enter** - 在输入框中按 Enter 快速添加任务
- **点击外部** - 点击输入框外部自动关闭输入框

## 视觉效果

- **浅色主题** - 浅色半透明背景
- **平滑动画** - 任务添加、完成、删除都有流畅动画
- **毛玻璃效果** - 背景模糊效果
- **极简设计** - 无多余元素

## 常见问题

### Q: 如何让窗口始终显示在最前面？
A: 点击左上角的绿色按钮，按钮变紫色表示已固定。

### Q: 为什么窗口不在 Alt+Tab 中显示？
A: 这是设计特性，让待办清单像桌面小部件，不干扰应用切换。

### Q: 任务数据会丢失吗？
A: 不会，所有数据自动保存在本地文件中。

### Q: 如何更改窗口位置？
A: 按住窗口标题栏（顶部区域）拖动即可。
